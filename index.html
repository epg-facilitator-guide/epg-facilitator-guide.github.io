How to run an ePG, in four steps

The whole job, end to end. Each step is detailed on page 2.

1

Create a class

You'll need to create a class each time you're at a dealership and want to track an individual ePG's completion. One class per store keeps the reporting clean.

2

Get the Associates in

Invite associates into the class via QR code or by sharing the link. They scan, they're in.

3

Start with Resources

Open the Resources tab and play the instructional video or guide first. It's the preamble — it frames the topic before the team dives into the Journey activities.

4

Run the ePG & track progress

Walk the team through the exercises in the app, have everyone complete the exam, and watch progress roll in via the facilitator view (called out on page 2).

Suggested run of show

A 30–45 minute in-person ePG, paced for one Journey + Exam.

0:00
Pre-class

Set up & share

Display the QR. Greet the team as they arrive. Make sure everyone has the app open and joins the class before you begin.

0:05
Frame

Why we're here

30-second framing: today's topic, why it matters at the dealership, and what success looks like by the end.

0:08
Resources

Open Resources & play the setup video

Open the Resources tab and play the instructional video together — on the room display or on phones. This is the preamble that frames the topic.

0:18
Practice

Run the role-plays

Pair up. Walk through the scenarios in the interaction Guide. Switch roles. Listen for the customer-friendly explanation.

0:35
Prove it

Take the exam

5 questions, 90% to pass. Anyone who doesn't pass retakes — that's how we know everyone's ready for the floor.

Facilitator tips

!

Generate a fresh code per dealership

Each store gets its own class. That's how completion data stays clean and reportable.

★

Lead the role-play yourself first

Model the customer-friendly explanation once. It sets the bar — and breaks the ice.

⟳

If someone can't join

Tap Generate New Code. The old code is invalidated — re-share the new one with the room.

✓

Check completions live

The Completions tab on the Class screen updates in real time. Use it to confirm everyone passed.

▶

Don't skip the role-play

The video teaches the concept. The role-play is what actually changes the showroom conversation.

⚐

End on the success commitment

Have each learner say one thing they'll do differently this week. Public commitments stick.

After the class

Review your completions from Class › Completions. Associates must complete all ePG content to reflect as ‘Complete’. Associates have unlimited retakes on the exam. SET leadership sees the same dashboard, so what you log here is what gets reported.

Step 1

Create a new class

Four screens, ~60 seconds. Do this before the team arrives — once the class exists, you're ready to invite.

When
5–10 min before class
1
Explore › Collection

Open the topic & tap Facilitate

Find the BEV Champion topic you're running. Tap Facilitate in the top-right of the collection screen.

2
Facilitate Class

Tap + Create New Class

You'll see your existing classes here. For a new event, tap + Create New Class at the top.

3
Create New Class

Name it & pick the dates

Use the dealership name (e.g. Atlanta Toyota). Select start and end dates, then tap Add Class.

4
Class › Overview

You're in. Class is live.

Each class gets a 6-character code (e.g. ZIKBKZ) and a QR. Both unlock the course for your team — go to Step 2 to share.

Pro tip

Generate a fresh class per dealership. That's how completion data stays clean — SET leadership sees one row per store.

Step 2

Invite participants

Two ways to get the team in: scan the QR in the room, or send the link via iMessage. Pick whichever fits your setup.

When
As the team arrives
1
Class › Overview

Show the unlock code & QR

From the Class screen, hold your phone up so the room can scan the QR code, or read out the 6-character unlock code for anyone who'd rather type it in.

2
Share Course

Or tap Share

Need to invite someone who isn't in the room? Tap Share — you'll get the standard iOS share sheet for iMessage, AirDrop, or email. Copy Link drops it on your clipboard.

3
iMessage / AirDrop

Send the link

Pick a contact (or a group thread for the whole team). The link opens straight into the RockED app and joins them to your class — no code needed.

Heads up

If a code stops working — say someone shared it outside the dealership — tap Generate New Code on the Class Overview. The old one is invalidated immediately; just re-share the new one with the room.

Step 3

Start with Resources

The preamble. Open the Resources tab and play the instructional video or guide that frames the topic — do this before the team opens the Journey activities.

When
Right after everyone joins
Set up the room

Before you play anything, connect a projector or big screen and cast the Resources material to it. Watch the setup video together as a group on the shared display rather than having everyone follow along on their own phones — it keeps the room in sync and starts the conversation as a team.

1
Collection › Content

Find the new Resources tab

On the collection screen you'll now see a Resources tab beside Content. Tap it to open the prep materials for this topic.

2
Collection › Resources

Play the resource first

Tap the video or guide under All Resources and watch it together as a group. This is the setup that frames the session before anyone starts the Journey.

Why first

Resources are the preamble. Everyone should watch the setup video before opening the Journey activities — it gets the whole room on the same page about why the topic matters.

Step 4

Track progress

As the team works through the ePG, the Completions tab updates live. Three buckets: Not Started, In Progress, Completed.

When
During & after class
1
Class › Completions

Tap the Completions tab

From the Class screen, switch from Overview to Completions. You'll see three buckets — Completed · In Progress · Not Started — with live counts.

2
In Progress

Watch the room move

Tap In Progress to see who's mid-flight — anyone who's started a Guide but hasn't finished the exam. Names appear as they begin.

3
Learner view

They finish & pass

When a learner clears the exam at 90%+, they see this. On your end, they shift from In Progress to Completed — and the dealership's reporting row updates.

Don't forget

Wrap up before anyone leaves. Glance at Completed — if the count matches the room, you're done. Anyone in In Progress at the end gets a quick coaching moment and a retake. Not Started = they probably never joined the class; check their unlock code.
